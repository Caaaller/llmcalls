## Setup

1. Install dependencies:
```bash
cd frontend
npm install
```

2. Start the development server:
```bash
npm start
```

The app will run on http://localhost:3001

## Backend API

Make sure the backend server is running on http://localhost:3000

## Usage

1. Enter the phone number to call (e.g., eBay support)
2. Enter the transfer phone number (where calls will be transferred)
3. Configure call purpose and custom instructions
4. Select voice option
5. View/edit the transfer prompt
6. Save settings or initiate a call
