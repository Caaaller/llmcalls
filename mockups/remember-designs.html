Recommendation
Combine A + C

Always auto-fill the phone number (Option A — simplest, covers the core ask). Plus show recent calls on Step 1 (Option C — enables fast repeat calls). This covers both needs without overcomplicating every step like Option B would.

  • Phone number saved to localStorage after each call — restored on Step 3
  • Recent calls already exist in the API — surface them on Step 1 with "skip to review"
  • Minimal new state management needed
